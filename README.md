A professional, ad-free, and open-source instrument tuner for Android. This app provides high-precision tuning for a variety of stringed instruments using real-time audio analysis.

## Description

**Guitar Tuner Pro** is designed for musicians who need a reliable, fast, and completely free tuning solution. Unlike many other tuners, this app contains **no advertisements** and respects your privacy. It features a clean, modern interface built with Jetpack Compose and a visual needle gauge to help you achieve perfect pitch.

The app is built with a modular architecture, making it easy to add new instruments or custom tuning profiles.

### Key Features
- **Keep Screen Awake**: The display stays on while you are tuning, so you don't have to keep touching the screen.
- **Multiple Instruments**:
  - **Guitar**: Standard, Drop D, and Open D5 (D-A-D-A-D-D).
  - **Bass Guitar**: Standard 4-string tuning.
  - **Ukulele**: Standard GCEA tuning.
  - **Banjo**: Standard 5-string tuning.
- **Visual Feedback**: High-precision gauge showing cents difference from the target note.
- **Frequency Display**: Real-time Hz readout for advanced users.
